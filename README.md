------------Git-------------
*git add .:* 			agregar archivos
*git reset .:*			revertir el git add
*git commit -m "commit":* 	agregar commits o fotografias
*git branch:*			ver las ramas existentes del repositorio
*git status:*			revisar que archivos no han sido commiteados
*git diff:*			revisar las diferencias que se han hechoen archivos no commiteados
*git log:*			ver historial de commits
*git restore:*			para descartar cambios en archivos en el que no se hayan hecho commits
*gith checkout:*		para mover de una rama a otra
*git checkout -b:*		para crear una rama y moverte a ella en una sola linea de comando
*git merge:*			fusionar la rama en la que estas con la que le especifiques
*git push -u origin:*		subir rama local a repositorio remoto
*git branch -d:	*		eliminar rama local
*git branch -D:*			forzar el borrado de la rama
*git push origin --delete:*	eliminar rama remota
*git checkout -- .:*		reconstuir al ultimo commit
